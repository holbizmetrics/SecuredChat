P2P Private Chat

Direct browser-to-browser messaging. No server. No account.

How does this work?

  1. One person creates a chat room and gets a code.
  2. They send that code to their friend (via email, text, etc.).
  3. The friend joins by pasting the code, and gets a code back.
  4. They send that code back to the first person.
  5. The first person pastes it, and you're connected!
Both people must have this page open at the same time. The codes are long — always use the Copy button, don't try to type them.
Messages travel directly between your browsers, encrypted in transit. No server ever sees them. However, this does not verify your friend's identity — make sure you share the codes through a channel you trust.
Messages support Markdown formatting: **bold**, *italic*, `code`, lists, tables, images, and even math with $\LaTeX$ syntax. You can toggle between rendered and raw view in the chat. Once connected, you can also make audio calls using the phone button.
